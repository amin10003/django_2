# django_2
create a .env file and add the following content as required

- SECRET_KEY= "paste you secret from the django project settings.py file"
- DATABASE_NAME="name of your database"
- DATABASE_USER="database user name"
- DATABASE_PASSWORD="your database password"
- DATABASE_HOST="your database host"
- DATABASE_PORT="your database port"
- CLOUDINARY_CLOUD_NAME="your cloudinary cloud name"
- CLOUDINARY_API_KEY="your created project api key"
- CLOUDINARY_API_SECRET="your created project secret key"
